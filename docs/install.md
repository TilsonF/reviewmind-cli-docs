# Instalacion

## Via npm (recomendado)

```bash
npm install -g remi-cli
```

## Uso local (repo)

```bash
git clone https://github.com/TilsonF/reviewmind-cli
cd reviewmind-cli
npm install
npm link
```

## Variables utiles

- `REMI_API_URL` para definir la URL de la API.
- `REMI_REFRESH_COOKIE_NAME` si cambias el nombre de la cookie.
