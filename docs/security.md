# Seguridad

## Tokens locales

- `remi login` guarda un access token y refresh token en `~/.remi/config.json`.
- Si compartes el equipo, protege ese archivo.

## Recomendaciones

- Usa un usuario con permisos minimos en ReviewMind.
- Rota el token si sospechas exposicion.
